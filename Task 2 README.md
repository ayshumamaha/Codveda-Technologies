# Number Guessing Game

## Overview

The Number Guessing Game is an interactive Python game where the system generates a random number and challenges the user to guess it correctly. The game provides hints to guide users toward the correct answer.

---

## Features

* Random number generation
* Interactive gameplay
* Hint system
* Win/Loss detection
* Multiple attempts

---

## Technologies Used

* Python 3.x
* Random Module
* Loops
* Conditional Statements
