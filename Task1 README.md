# Simple Calculator

## Overview

The Simple Calculator is a Python-based application designed to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. The project provides a simple and interactive command-line interface that allows users to perform calculations quickly and efficiently.

This project demonstrates fundamental Python programming concepts including functions, conditional statements, user input handling, and mathematical operations.
