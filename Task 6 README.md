# N-Queens Problem Solver

## Overview

The N-Queens Problem Solver is a Python application that solves the classic N-Queens puzzle using backtracking algorithms. The objective is to place N queens on a chessboard such that no two queens attack each other.

This project demonstrates algorithm design, recursion, and optimization techniques.

---

## Features

* N-Queens puzzle solving
* Backtracking algorithm implementation
* Chessboard visualization
* Multiple solution generation
* Efficient conflict checking
